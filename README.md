# 26-Xi-quizbank

习思想复习题库 — 习近平新时代中国特色社会主义思想在线刷题工具

## 功能

- 📝 **完整题库**（142 题：单选 72 + 多选 70），随机排序
- 🎯 **错题集**（自动筛选错误率 > 30% 的题目）
- 📊 **作答统计**（每题正确率、错误率、作答次数）
- 📋 **作答记录**（时间、答案、对错）
- ⌨️ **键盘操作**：1-4 选选项 · Enter 提交 · ← → 翻题
- 💾 **本地持久化**（localStorage），导出/导入 JSON 备份

## 文件说明

| 文件 | 说明 |
|------|------|
| `习思想刷题.html` | 主程序（浏览器直接打开） |
| `parse_questions.py` | PDF 题库解析脚本 |
| `questions.json` | 解析后的题库数据（142 题） |
| `.gitignore` | Git 忽略规则 |

## 使用方式

1. 用浏览器打开 `习思想刷题.html`
2. 首页选择「完整题库」或「错题集」开始刷题
3. 单选/多选完成后点「提交本题」判分
4. 可在首页导出备份 JSON 文件，下次自动加载

## 备份机制

- 导出：首页点「📥 导出备份」，保存为 `习思想题库备份.json`
- 加载：打开页面时自动检测同目录备份文件并加载
- 也可手动「🔄 加载备份」或「📤 选择文件导入」
