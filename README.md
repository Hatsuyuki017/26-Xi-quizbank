# 26-Xi-quizbank

思政复习题库 — 毛泽东思想 + 习近平新时代中国特色社会主义思想在线刷题工具

## 功能

### 答题
- 📝 **双学科题库**：毛泽东思想（245 题：单选 78 + 多选 84 + 判断 83）+ 习近平思想（142 题：单选 72 + 多选 70）
- 📋 **顺序/乱序**：毛概支持按章节顺序作答，习概乱序作答
- 🏷️ **题型选项卡**：进度条形式实时显示各题型（单/多/判）已答/总数，点击跳转并记住位置
- 🏁 **结束作答**：保留当前轮次所有作答状态，结束后统一结算本轮正确率
- 📝 **笔记栏**：答题页右侧笔记区，提交题目后可编辑，纯文本格式，跨页面持久化
- ⌨️ **键盘操作**：1-4 选选项 · Enter 提交/下一题 · ← → 翻题

### 记录与分析
- 🎯 **错题集**（自动筛选错误率 > 30% 的题目）
- 📊 **作答统计**（每题正确率、错误率、作答次数），点击行查看题目详情及各选项累积选择频率
- 📋 **作答记录**（时间、答案、对错），点击行回顾该次作答详情
- 📈 **正确率趋势**：首页点击正确率 → SVG 折线图，每日正确率 + 分题型趋势
- 💾 **本地持久化**（localStorage），导出/导入 JSON 备份（含笔记）

### 视觉
- 🔢 Palatino 数字字体
- 📊 累计已答进度条 + 各题型独立进度条
- 🎨 **25 套配色方案**（Thomas Shang's Tufte-Style Book Template 24 palettes + warm），悬停预览、点击切换，持久化保存

## 使用方式

1. 用浏览器打开 `习思想刷题.html`
2. 首页选择学科和模式（完整题库 / 错题集）
3. 毛概可选「按章节顺序」或「乱序」作答
4. 通过题型选项卡在单选/多选/判断间自由切换
5. 提交题目后在右侧笔记栏记录要点
6. 点击「结束作答」查看本轮正确率统计
7. 在「作答统计」「作答记录」中点击行查看题目详情
8. 首页点击正确率查看 SVG 趋势图
9. 导航栏「🎨 主题」切换配色方案

## 备份机制

- 导出：首页点「📥 导出备份」，保存为 `思政题库备份.json`
- 加载：打开页面时自动检测同目录备份文件并加载
- 也可手动「🔄 加载备份」或「📤 选择文件导入」
